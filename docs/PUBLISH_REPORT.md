## ✨ 用户使用方式

### 通过 pip 安装

```bash
pip install glossmod-mcp
```

### 在 Claude Desktop 中配置

编辑 `~/.claude_desktop_config.json`：

```json
{
  "mcpServers": {
    "glossmod": {
      "command": "uv",
      "args": [
        "run",
        "--with",
        "glossmod-mcp",
        "python",
        "-m",
        "glossmod_mcp.server"
      ],
      "env": {
        "GLOSSMOD_API_KEY": "your-api-key"
      }
    }
  }
}
```
